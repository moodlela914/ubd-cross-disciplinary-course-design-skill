# UBD 跨域課程設計 Skill

這是一個可重複使用的 ChatGPT / Codex Skill，協助教師依據臺灣十二年國民基本教育課程綱要，運用 UbD 逆向教學設計、PBL 專題導向學習、K-U-D 學習目標、GRASPS 表現任務與 Rubrics 評量規準，快速產出跨領域素養導向課程設計。

---

## 專案目標

本 Skill 的目標是協助教師將抽象的跨領域主題，轉化為具體、可教、可評量、可實作的課程方案。

它特別適合用於：

- 跨領域課程設計
- SDGs 議題融入教學
- 108課綱素養導向教案
- UbD 逆向教學設計
- PBL 專題導向學習
- 教學活動流程設計
- 評量規準設計
- Word 教案格式填寫前置規劃

---

## 核心功能

- 依據年級、節數與跨領域科目產生課程設計
- 建立課程大概念 Big Idea
- 連結真實情境與 SDGs 議題
- 設計 K-U-D 學習目標
- 設計 GRASPS 表現任務
- 產出 Rubrics 評量規準
- 產出 5 至 10 分鐘切分的教學流程表
- 提供差異化與延伸學習建議
- 支援轉換為 Word 教案格式

---

## 必要輸入

使用本 Skill 前，建議先提供以下三項資訊：

1. 適用年級  
2. 課程總節數  
3. 跨領域科目、學習領域或 SDGs 議題

# PBL 專題導向學習備忘

PBL，Project-Based Learning，強調學生透過真實問題、探究歷程與公開成果，建立知識、技能與素養。

---

## 一、PBL 核心精神

PBL 不只是做作品，而是透過真實問題引導學生進行持續探究，最後產出可以被他人理解、評估或使用的成果。

---

## 二、PBL 重要元素

| 元素 | 說明 |
| :--- | :--- |
| 真實問題 | 來自生活、社會、環境或真實世界需求 |
| 驅動問題 | 能引發探究，不是單一標準答案 |
| 持續探究 | 學生需要蒐集資料、分析、修正想法 |
| 學生選擇 | 學生在主題、方法或產出上有一定選擇權 |
| 公開成果 | 學習成果可以分享給真實或模擬觀眾 |
| 反思修正 | 學生根據回饋改善作品或想法 |
| 合作學習 | 學生透過分工與討論共同完成任務 |

---

## 三、驅動問題設計

好的驅動問題應該：

- 開放
- 真實
- 有挑戰
- 能連結多個學科
- 能導向作品或行動
- 能引發學生想知道更多

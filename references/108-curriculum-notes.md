# 十二年國教 108 課綱使用備忘

本文件提供本 Skill 在生成課程設計時使用的課綱處理原則。

---

## 一、使用原則

本 Skill 可協助教師對應十二年國民基本教育課程綱要，但正式課綱代碼與學習重點仍應以教育部公告版本為準。

生成課程時，應優先處理：

1. 核心素養方向
2. 學習表現方向
3. 學習內容方向
4. 教學活動與評量任務對應
5. 實際課堂可操作性

---

## 二、核心素養三面九項

十二年國教核心素養可分為三大面向與九個項目。

### A 自主行動

- A1 身心素質與自我精進
- A2 系統思考與解決問題
- A3 規劃執行與創新應變

### B 溝通互動

- B1 符號運用與溝通表達
- B2 科技資訊與媒體素養
- B3 藝術涵養與美感素養

### C 社會參與

- C1 道德實踐與公民意識
- C2 人際關係與團隊合作
- C3 多元文化與國際理解

---

## 三、常見跨域課程對應方向

| 課程類型 | 常見核心素養方向 |
| :--- | :--- |
| 問題解決導向課程 | A2、A3 |
| 專題探究課程 | A2、A3、B2 |
| 數位工具融入課程 | B2、B1 |
| 藝術創作課程 | B3、B1 |
| 小組合作課程 | C2、B1 |
| 社會議題課程 | C1、C3、A2 |
| 國際議題課程 | C3、B1 |
| 永續發展課程 | C1、A2、A3 |
